# SCPSL-GOC-Plugin
在对局中添加了GOC阵营

## 插件介绍:
  当回合开始20分钟后,会在办公A电梯生成由8个玩家组成的一支GOC攻击小组,与其他阵营敌对,GOC人员在核弹室抛硬币可以启动Omega核弹,倒计时90秒,核弹在关闭后可以再次开启但倒计时不变\n
  GOC由1个指挥官,2个重装兵,和5个士兵组成\n
  指挥官有1把电炮,狗官枪,重甲,蓝苹果,黑卡,150的血量上限,肾上腺素,500,120的抗性0.75的AHP,SCP-1853效果\n
  重装兵有1把3x,混沌大机枪,重甲,混沌卡,200的血量上限,肾上腺素,500,200的抗性0.85的AHP,强度30的缓慢效果\n
  士兵有1把囚鸟,中士枪,中甲,混沌卡,120的血量上限,肾上腺素,500,100的抗性0.7的AHP,强度35的加速效果\n
  Omega核弹只能由GOC人员开启,倒计时结束后杀死所有人\n
